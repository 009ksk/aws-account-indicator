# アイコンファイルについて

この拡張機能を完全に動作させるには、以下のアイコンファイルが必要です：

## 必要なファイル

- `icon16.png` (16x16ピクセル) - ツールバー用小アイコン
- `icon48.png` (48x48ピクセル) - 拡張機能管理画面用アイコン
- `icon128.png` (128x128ピクセル) - Chrome ウェブストア用アイコン

## 推奨デザイン

アイコンは以下の要素を含むことを推奨します：

- AWS関連のシンボル（クラウド、AWSロゴ風の要素など）
- 複数色やレイヤーを表現する要素（アカウント識別の意味）
- オレンジ系の色調（拡張機能のテーマカラーに合わせて）

## 作成方法

1. **オンラインツール**: [Canva](https://www.canva.com)、[GIMP](https://www.gimp.org/)などを使用
2. **AI生成**: MidJourney、DALL-E、Stable Diffusionなどで生成
3. **既存素材**: [Flaticon](https://www.flaticon.com/)、[Icons8](https://icons8.com/)などから取得

## 作成後

1. 各サイズのPNGファイルを作成
2. このフォルダ（`icons/`）に配置
3. ファイル名を正確に合わせる：
   - `icon16.png`
   - `icon48.png` 
   - `icon128.png`

## 注意事項

- 透明背景を推奨
- 高コントラストで視認性を重視
- Chrome拡張機能のガイドラインに準拠
